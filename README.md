# icefelix_window_manager

⚠️ **Work in progress** — v0.1.0-dev. Public API stable when v0.1.0 ships.

- ✅ W1: Dart foundation + Pigeon schema (2026-05-22)
- ⏳ W2: macOS native impl
- ⏳ W3: Windows native impl
- ⏳ W4: Linux native impl (libdecor)
