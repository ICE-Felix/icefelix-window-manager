# icefelix_window_manager_platform_interface

Platform interface for [`icefelix_window_manager`](https://pub.dev/packages/icefelix_window_manager).

App developers: **depend on `icefelix_window_manager`**, not this package.

Platform implementers: extend `WindowManagerPlatform` and wire Pigeon-generated `WindowHostApi` to native code.
